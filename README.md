# Multi_Signal_Visualizer

## Features
1. Multiple signals
2. Realtime plotting
3. Easy navigation and zooming
4. Colour change, Legend renaming
5. Speed control
6. Export to pdf

## Accepted Formats
csv, txt, xls, hea, dat, rec

## Libraries

- PyQt5
- pyqtgraph
- fdpf
- pandas
- matplotlib
- wfdb
